# Airbnb Clone

A full-stack Airbnb clone built from scratch as a practice project. While visually it might be a clone or similar to that of AirBnB, the tech stack will differ.

## Overview

This project replicates core Airbnb functionality including property listings, search and filtering, booking management, and user authentication. Built with a focus on clean architecture and user experience.

## Features

### Planned Core Features
- **User Authentication**: Sign up, login, and profile management
- **Property Listings**: Browse and search available properties
- **Advanced Search**: Filter by location, dates, price, amenities, and guests
- **Property Details**: View detailed information, photos, and reviews
- **Booking System**: Reserve properties with date selection and pricing
- **Host Dashboard**: List and manage properties
- **Reviews & Ratings**: Leave and view property reviews
- **Favorites**: Save properties to wishlist
- **Payment Integration**: Secure payment processing
- **Responsive Design**: Mobile-first approach

## Tech Stack

**Note**: This tech stack differs from Airbnb's actual implementation.

### Frontend
- **Next.js** (React + TypeScript)
- **Tailwind CSS**

### Backend
- **Fastify** (TypeScript) 
- **Zod**

### Database
- **PostgreSQL**

### Authentication
- **Supabase Auth**

### Additional Tools
- **Supabase Storage** - File/image storage
- **Maps Integration**
- **Stripe** - Payment processing (test mode for development with fake payments)
